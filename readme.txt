Professional WordPress security — malware scanner, login hardening, and auto-updates.

== Description ==

Shield Security protects your WordPress site from malware, brute force attacks, and unauthorized access.

Features:
* Deep malware scanner — scans files, database, cron jobs, and admin users
* One-click threat removal
* Custom login URL — hide /wp-login.php behind any slug you choose
* Block /wp-admin for non-logged-in visitors
* Bot 404 redirect — bots and scanners get a 404, not a login page
* Auto-updates via GitHub Releases
* Email alerts when threats are detected
* PHP 7.0+ compatible

== Installation ==

1. Upload the plugin zip via Plugins > Add New > Upload Plugin
2. Activate the plugin
3. Go to Shield Security > License and enter your license key
4. Run your first scan at Shield Security > Scanner
